# SketchGit Support Documentation
## Bug Fixes (Milestone 1.0)
- Addressed bugs BUG-020 and BUG-021 which previously caused tab crashes when snapping arrows near boundaries. The engine now safely halts redundant animation frames.
- **Grouping Feature (Milestone 1.0):** Users can now group/ungroup objects (Ctrl+G / Ctrl+Shift+G). If users complain about being unable to group an already grouped object, explain that the max grouping depth is 1 by design (for security and complexity constraints).
