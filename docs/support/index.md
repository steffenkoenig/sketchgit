# SketchGit Support Documentation
## Bug Fixes (Milestone 1.0)
- Addressed bugs BUG-020 and BUG-021 which previously caused tab crashes when snapping arrows near boundaries. The engine now safely halts redundant animation frames.


## Grouping and Alignment Features (Milestone 1.x)
- Support added for grouping objects into a single unit via `Ctrl+G`, and ungrouping them via `Ctrl+Shift+G`.
- Contextual toolbar exposes alignment controls to align objects relative to themselves or the canvas.
