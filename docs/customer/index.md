# SketchGit Customer Documentation
## GDPR Compliance Update (Milestone 1.0)
SketchGit no longer relies on Google Fonts CDN. All fonts are self-hosted to ensure GDPR compliance (GAP-016).


## Grouping and Alignment (Milestone 1.x)
SketchGit now supports Object Grouping and Alignment Tools. You can easily snap your architectural blocks, UI components, and text labels into perfect alignment with a single click. Group elements together to treat complex compositions as a single asset using `Ctrl+G` (Group) and `Ctrl+Shift+G` (Ungroup).
