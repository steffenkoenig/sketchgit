# SketchGit Customer Documentation
## GDPR Compliance Update (Milestone 1.0)
SketchGit no longer relies on Google Fonts CDN. All fonts are self-hosted to ensure GDPR compliance (GAP-016).
## Object Grouping and Alignment (Milestone 1.0)
SketchGit now supports grouping and aligning multiple objects directly on the canvas!
- **Grouping**: Select multiple objects and use the `Ctrl+G` (or `Cmd+G`) keyboard shortcut, or click the Group button in the properties panel. You can treat complex compositions as a single asset.
- **Ungrouping**: Select a grouped object and use `Ctrl+Shift+G` (or `Cmd+Shift+G`), or click the Ungroup button. Note: Grouping depth is limited to 1 for security and simplicity.
- **Alignment**: Select multiple objects and use the alignment tools in the properties panel to easily align them to the Left, Right, Top, Bottom, or Center.
